## Descobrindo Intervalos

Há intervalos que aparecem na música em todo o mundo. De onde eles vêm? Eles vêm de uma estrutura na natureza que é chamada de série harmônica na qual cada frequência é um múltiplo inteiro de uma fundamental. Vamos explorar a série harmônica e encontrar alguns sons antigos e novos.

### Complexidade do intervalo

Os intervalos são nomeados em relação ao fundamental. Neste caso, a fundamental pode ser descrita pela operação `Fm 1`. O intervalo simples é uníssono. Fm 1 soando ao mesmo tempo que Fm 1. Seguindo o uníssono é a duplicação (Fm 2) e metade das frequências (Fm 1/2) que vimos no último tutorial. Uma frequência dobrada é conhecida como uma oitava na teoria musical ocidental (TMO). Outra maneira de dizer que a distância entre o 1º e o 2º harmônico é uma oitava.

What about the distance between the 2nd and 3rd overtones? The name in TMO is the `Perfect 5th`. Listen below to the interval. First in the natural position in the overtone series and then the ratio transposed down an octave so we can here it in the context of the fundamental, `Fm 1`.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Seq [
  -- First in their natural position in the overtone series.
    Fm 2, 
    Fm 3, 
    Fm 2, 
  -- And now we will use the interval directly -- 3/2 -- which will sound in
  -- the same octave as the fundamental. 
    Fm 1, 
    Fm 3/2, 
    Fm 1, 
  ]
}`
}/>

### Major Scale

We can discover all sorts of beautiful and useful intervals by exploring the distances between the natural overtone series.


Here's an C Major Scale (261.6hz is an A) built using ratios from the overtone series. The first note is the fundamental. The second is the distance between the 9th and 8th. The third interval is the distance between the 5th and 4th overtones...etc...

<WereSoCool language={
`{ f: 261.6, l: 1/3, g: 1, p: 0 }\n
main = {
  Seq [
    Fm 1, Fm 9/8, Fm 5/4, Fm 4/3, 
    Fm 3/2, Fm 5/3, Fm 15/8, Fm 2
  ]
}`
}/>
