## Notación de lista: `&` 
Las listas se indican con un `&`. Puede indexar la lista con una `@` seguida de `[operaciones]` Las listas no admiten índices negativos.

<WereSoCool
  readOnly
  hideGutter
  language={`&[op1, op2, op3, ...] @ [indices]`}
/>

### Lista Básica

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
-- Los índices de la lista comienzan en 0
main = {
  &[Fm 1, Fm 9/8] @ [0, 1, 0]
}`
}/>


### Listas y Aleatorias

<WereSoCool language={
`{ f: 261.6, l: 1/4, g: 1, p: 0 }\n
-- También podemos elegir índices aleatoriamente.
-- Cada vez que lo juegues, será diferente.
main = {
  &[Fm 0, Fm 1, Fm 9/8, Fm 3/2] @ [Random(8)]
}`
}/>

### Una Composición Usando Listas

<WereSoCool language={
`{ f: 285, l: 1, g: 1/5, p: 0 }\n
overtones = {
  Overlay[
    (2/1, -3, 1, 1),
    {2/1, 0, 1, -1},
    {1/1, 3, 1, 1},
    {1/1, 0, 1, -1},
    {1/2, 0, 1, 1},
    {1/2, 3, 1, -1},
  ]
}\n
-- Esta es una declaración de lista.
-- En esta lista, encontrará una escala mayor con
-- con un silencio (Fm 0) al final.
cool_list = &[
  Fm 1, Fm 9/8, Fm 5/4, Fm 4/3,
  Fm 3/2, Fm 5/3, Fm 15/8, Fm 2,
  Fm 0,
]\n 
main = {
  overtones
  | Seq [
    -- Primero en la secuencia es la lista reproducida como
    -- una secuencia.
    &cool_list,
    -- A continuación, la lista indexada en. La sintaxis para los índices es:
    -- &nombre_lista @ [índice1, índice2, índice3, etc.]
    &cool_list @ [
      0, 1, 2, 3, 2, 1, 3, 5, 
      7, 6, 5, 4, 3, 2, 1, 0, 8
    ],
    -- Hay algunos tipos especiales de índices en el lenguaje.
    -- Random(n) es un tipo especial de índice que
    -- elige n elementos aleatorios de tu lista
    &cool_list @ [Random(30)] | Lm 2/3,
    -- Y aquí hay una pequeña melodía para terminar nuestra frase.
    Seq [
      Fm 3/2,
      Fm 15/16, 
      Fm 1
    ] 
    | Lm 3
  ]
  | Lm 1/7
}`
}/>
