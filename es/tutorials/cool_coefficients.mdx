### Múltiplos

-1, -1, -1, -1, -1, -1, -1. 

Eso es algo extraño de escribir. Somos tan geniales
tiene una abreviatura para múltiples coeficientes seguidos en
un generador Puedes usar `*` para decir cuántos de un coeficiente
debe suceder en una fila,

<WereSoCool language={
`{ f: 261.6, l: 1/10, g: 1, p: 0 }\n
main = {
  *(
    -- esencialmente, sube 4 medios pasos 3 veces
    -- luego baja 5 medios pasos 3 veces
    -- sigue haciéndolo hasta que llegues a los 40
    f: 0|12 [4*3, -5*3]
  ) Take 40
  | Repeat 3
}`
}/>

### Expresiones Matemáticas
Ambas son sintaxis experimentales. bit de "inicio" y "subdivisión" de forma diferente.
Para todos los ejes, Poly o Expr se aplica empezando por el inicio 
y avanzando paso a paso a través de las subdivisiones. 
Entonces, 9|10 comenzaría en 9/10 y luego
los siguientes valores serían el resultado de la expresión aplicada
al 10/10 seguido de la expresión aplicada al 11/10. en la frecuencia
eje, el resultado se normaliza al espacio de registro.

### Poli - Polinomios
Crea un polinomio a partir de una lista de coeficientes en orden ascendente.

<WereSoCool language={
`{ f: 261.6, l: 1/10, g: 1, p: 0 }\n
main = {
  *(
    -- -1/3x^2 + x + 2
    f: 0|12 Poly [2, 1, -1/3]
  ) Take 70
}`
}/>

### Expr - Expresiones

#### Operaciones
> +, -, *, /

#### Funciones
> sqrt, abs

> exp, ln, log10

> sin, cos, tan, asin, acos, atan, atan2

> sinh, cosh, tanh, asinh, acosh, atanh

#### Constantes
> pi, e

<WereSoCool language={
`{ f: 261.6, l: 1/10, g: 1, p: 0 }\n
main = {
  *(
    f: 0|12 Expr \`sin(5 * x)\`
  ) Take 70
}`
}/>
