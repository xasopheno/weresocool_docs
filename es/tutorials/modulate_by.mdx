## `Operación: ModBy` 
`ModBy` te permite usar una operación para modular otra operación.

<WereSoCool
  readOnly
  hideGutter
  language={`operacion | ModBy [operaciones]`}
/>

<WereSoCool language={
`{ f: 261.6, l: 1/6, g: 1, p: 0 }\n
melodia = {
  Seq [Fm 1, Fm 9/8, Fm 5/4, Fm 3/2]
}\n
main = {
  Seq [
    Seq [
      melodia,
      melodia
      | ModBy [Fm 1, Fm 1/2],
      melodia
      | ModBy [Fm 1, Gm 1/3],
      melodia
      | ModBy [Fm 1, Overlay [Fm 1, Fm 3/2]],
      melodia
      | ModBy [Fm 4/3, Fm 1] 
      | Overlay [Fm 1, Fm 3/2, Fm 1/2],
      melodia
      | Overlay [Fm 1 | Pa 2/10, Fm 5/4 | Pa -1/10, Fm 1/2]
      | Reverse
    ],
  ]
}`
}/>

---

`ModBy` es útil cuando has hecho algo genial
y luego quiero hacer cambiarlo dejando la longitud
lo mismo.

<WereSoCool language={
`{ f: 285, l: 1, g: 1/3, p: 0 }\n
overtones = {
  O[
    (1/1, 4, 1, 1),
    (1/1, 0, 1, -1),
  ]
}\n
-- Esta operación es esencialmente un desvanecimiento.
gain_modulator = {
  Seq [
    Gm 1, Gm 2/3, Gm 1/3, Gm 1/6, Gm 1/9, Gm 1/12
  ]
}\n
-- Y usaremos esta operación para modular la melodía.
melodic_modulator = {
  Seq [
    Fm 1,
    Overlay [Fm 5/4, Fm 3/2],
    Overlay [Fm 5/6, Fm 9/8, Fm 4/3] | Lm 2
  ]  
  | Repeat 6
}\n
-- Recomendaría comentar las cosas dentro y fuera
-- para averiguar cómo funciona esto.
thing1 = {
  Seq [
    -- una melodia simple
    Fm 1, Fm 4/3, Fm 5/3
  ]
  -- Modulada por las dos operaciones que creamos anteriormente.
  | ModBy [melodic_modulator]
  | ModBy [gain_modulator]
}\n
main = {
  overtones
  | thing1
  | Seq [Fm 1, Fm 9/8]
  | Seq [Fm 1, Fm 5/4]
}
`
}/>
