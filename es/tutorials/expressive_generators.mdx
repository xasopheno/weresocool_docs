Los generadores son geniales. Sigamos con ellos.

### Melodías infinitas

Vamos a permanecer en ET (12) por ahora. Usando generadores, nosotros
puede hacer algunas estructuras melódicas hermosas, largas y sinuosas. 

<WereSoCool language={
`{ f: 261.6, l: 1/8, g: 1, p: 0 }\n
-- Tres Melodías de generadores.
main = {
  Overlay [Fm 1/2, Fm 1, Fm 2]
  | Seq [
    *(
      f: 0|12 [7, -3, -4, -2, 1]
    ) Take 30,
    Fm 0,
    *(
      f: 0|12 [-3, 4, 3, 2, -4, 2, 1, -2]
    ) Take 25,
    Fm 0,
    *(
      f: 12|12 [2, -3, 4, -5, 8, -3, -2, 1, -2, 2, -3, -1, -1, 2]
    ) Take 60,
  ]
}`
}/>

# Tiempo se siente
Oh, a los músicos les encanta hablar sobre la sensación del tiempo. Uno de los
el mayor cumplido que le puedes dar a alguien.
"Hombre... ella tiene la mejor sensación de tiempo..."
Para un lenguaje de programación, creo que WereSoCool
tiene una muy buena sensación de tiempo. Los generadores son
un lugar muy agradable para explorar la relación
entre melodía, tempo y sensación de tiempo.

<WereSoCool language={
`{ f: 261.6, l: 1/5, g: 1, p: 0 }\n
-- Tres sensaciones de tiempo diferentes de los generadores.
main = {
  Overlay [Fm 1/2, Fm 1, Fm 2]
  | Seq [
    -- Swinging, pero relativamente recto
    *(
        f: 0|12 [7, -3, -4, -2, 1],
        l: 13|13 [-3, 3],
        -- Estos generadores de ganancia acentúan los upbeats
        g: 10|10 [-3, 3]
    ) Take 31,
    Fm 0,
    -- Bastante swinging
    *(
        f: 0|12 [7, -3, -4, -2, 1],
        l: 15|13 [-6, 6],
        g: 10|10 [-3, 3]
    ) Take 31,
    Fm 0,
    -- Muy swinging
    *(
        f: 0|12 [7, -3, -4, -2, 1],
        l: 16|13 [-8, 8],
        g: 10|10 [-3, 3]
    ) Take 31
  ]
}
`
}/>

# Dinámica
Cualquier forma fácil de agregar dinámica a algo que ya tienes
escrito es para ModBy [generador].

<WereSoCool language={
`{ f: 261.6, l: 1/6, g: 1, p: 0 }\n
main = {
  Overlay [Fm 1/2, Fm 1, Fm 2]
  | ET(12) @ [Random(35)]
  | ModBy [
    *(g: 10|10 [-1]) Take 10
  ]
}`
}/>
