Las operaciones de timbre cambian la forma de onda generada por un oscilador. Tenga cuidado al pasar de un timbre a otro en la misma voz, no siempre funcionará bien. Lo mejor es crear una nueva voz o agregar un espacio cuando desee cambiar el tipo de oscilador.

## `Operación: Sine`

La operación `Sine` aplica la siguiente fórmula a cada muestra. `Sine 1` es el sonido genial predeterminado de WereSoCool.

`sin(fase^potencia) / potencia`.


<WereSoCool
  readOnly
  hideGutter
  language={`--potencia debe ser un valor mayor que 0..\n
  Sine <potencia>`}
/>


<WereSoCool language={
`{ f: 161.6, l: 2, g: 1/2, p: 0 }\n

  Seq [Fm 1, Fm 0 | Lm 1/4]
  | Seq [
    Sine 1, 
    Sine 1.5, 
    Sine 2.0, 
    Sine 2.5, 
    Sine 3, 
    Sine 3.5,
    Sine 0.5,
    Sine 1, 
  ]
}`
}/>

## `Operación: Tri`
La operación `Tri` aplica la siguiente fórmula a cada muestra.
`(abs(sin(fase^potencia)) * 2 - 1) / potencia`

## `Operación: Square`

`Square` se puede considerar como una operación "pulse width". El valor en cualquier muestra dada se calcula con la
fórmula:

s = sin(fase);
si s > pulse_width { -1. } más { 1. };

El "pulse width" predeterminado es 0.5

<WereSoCool
  readOnly
  hideGutter
  language={`--El pulse_width debe tener un valor > 0 y < 1\n
  Square <pulse_width>`}
/>


<WereSoCool language={
`{ f: 161.6, l: 2, g: 1/2, p: 0 }\n
main = {
  Seq [Fm 1, Fm 0 | Lm 1/4]
  | Seq [
    Square 0.1, 
    Square 0.5, 
    Square 0.75, 
  ]
}`
}/>
