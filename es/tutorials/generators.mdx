Los generadores permiten describir infinitas listas en WereSoCool.

Un generador es una colección de Expresiones de eje. 
Los generadores comienzan con un `*`.

<WereSoCool
  readOnly
  hideGutter
  language={
`*(Expresiones de eje...) Take n`
}/>

### Expresiones de Eje
<WereSoCool
  readOnly
  hideGutter
  language={
`eje: inicio|subdivisión [coeficientes]`
}/>

Una expresión de eje tiene 4 partes: eje, inicio, subdivisión, coeficientes

### Eje
Hay cuatro ejes en WereSoCool:
Frecuencia, Longitud, Ganancia y Pan.
f, l, g y p en el contexto de un generador.

### Subdivisión
Una parte un poco complicada. Esto depende del eje. Para
Longitud, Ganancia y Pan, estamos dividiendo 1 en tantos
subdivisiones Para frecuencia, estamos dividiendo y octava en eso
muchas partes. Estamos creando un generador en el espacio ET(n).

### Comienzo
Nos vamos un poco fuera de secuencia ya que necesita saber qué
la subdivisión está haciendo antes de que pueda entender "inicio". "Inicio" dice: "Calcular el valor inicial para este eje
en este generador tomando el valor de entrada en ese eje y
agregando inicio * subdivisión". Hay algunos ejemplos a continuación que espero que esto quede más claro.

### Coeficientes
Los coeficientes describen el cambio en el eje en cada paso.
La lista de coeficientes se repite infinitamente.

## Generador de escala cromática

La siguiente es una expresión de eje para una
generador de escala cromática. Lo leo como:
En el eje de frecuencia, comience en la entrada
frecuencia * (0 * subdivisión). Cada paso, agregar
1 * subdivisión a la última frecuencia.

<WereSoCool
  readOnly
  hideGutter
  language={
`f: 0|12 [1]`
}/>

### Take

La última parte del Generador es Take. Desde generadores
son infinitos, debe ser explícito cuántos pasos
te gustaría tomar. Los generadores en realidad tienen algunos valores predeterminados
en su lugar y hará sonidos sin una toma, pero es
Siempre es mejor usar un Take.

Aquí usaremos nuestra expresión del eje de escala cromática
dentro de un Generador y luego Take 12.

<WereSoCool language={
`{ f: 311.127, l: 1/6, g: 1, p: 0 }\n
main = {
  *(
    f: 0|12 [1]
  ) Take 12
}`
}/>

## Más expresiones de eje

Para que quede claro lo que están haciendo los generadores,
vamos a añadir un pequeño ataque al principio de cada nota.

<WereSoCool language={
`{ f: 311.127, l: 1/6, g: 1, p: 0 }\n
main = {
  -- Normalmente describo el ataque de una nota como una Overlay
  -- de Fm 1 y luego una secuencia que describe el ataque.
  Overlay [
    Fm 1,
    -- Fm 2 is the frequency of the attack, the first 
    -- Fm 0 is the decay of that attack, and the second
    -- Fm 0 is the silence after the decay.
    Seq [Fm 2, Fm 0, Fm 0] | Lm 1/8,
  ]
  | Repeat 4
}`
}/>

### Escala disminuida

Una escala disminuida en ET(12) es una escala sintética construida a partir de
alternando pasos enteros y semitonos. Hay dos escalas disminuidas:
Entero-Medio y Medio-Entero. En WereSoCool podrías
Piense en ello como 2-1 y 1-2. Aquí hay una escala disminuida 2-1.

<WereSoCool language={
`{ f: 311.127, l: 1/6, g: 1, p: 0 }\n
main = {
  Overlay [
    Fm 1,
    Seq [Fm 2, Fm 0, Fm 0] | Lm 1/8,
  ]
  | *(
    f: 0|12 [2, 1]
  ) Take 9
}`
}/>

### Ganar
En una expresión de eje en el eje de ganancia, el valor resultante será Max(value, 0). No hay ganancias negativas. Puedes escribir expresiones que viajan
bajo cero, pero solo sonarán cuando el valor sea positivo

<WereSoCool language={
`{ f: 311.127, l: 1/5, g: 1, p: 0 }\n
main = {
  Overlay [
    Fm 1,
    Seq [Fm 2, Fm 0, Fm 0] | Lm 1/8,
  ]
  | *(
    g: 10|10 [-1],
  ) Take 10
  | Repeat 3
}`
}/>

### Longitud
En una expresión de eje en el eje de longitud, el valor más pequeño que se puede
generado es 1/subdivisión. Las longitudes menores o iguales a 0 no tienen sentido, por lo que la longitud será Máx. (valor, 1/subdivisión)

<WereSoCool language={
`{ f: 311.127, l: 1/2, g: 1, p: 0 }\n
main = {
  Overlay [
    Fm 1,
    Seq [Fm 2, Fm 0, Fm 0] | Lm 1/8,
  ]
  | *(
    l: 10|10 [-1],
  ) Take 10
  | Repeat 3
}`
}/>

### Pan
En un eje expresado en el eje panorámico, debe tener en cuenta que 0 es el centro panorámico.
Recomendé mantener los valores de pan entre -1 y 1
Los números positivos se desplazan hacia la izquierda y los negativos hacia la derecha.

<WereSoCool language={
`{ f: 311.127, l: 1/6, g: 1, p: 0 }\n
main = {
  Overlay [
    Fm 1,
    Seq [Fm 2, Fm 0, Fm 0] | Lm 1/8,
  ]
  | *(
    p: 5|5 [-1],
  ) Take 10
  | Repeat 3
}`
}/>

### Expresiones de Ejes Múltiples

Las cosas empiezan a ponerse bastante bien cuando
combine múltiples expresiones de eje en un solo generador.

<WereSoCool language={
`{ f: 311.127, l: 1/3, g: 1, p: 0 }\n
main = {
  Overlay [
    Seq [Fm 2, Fm 0 | Lm 8] | Lm 1/10,
    Fm 1,
    Fm 1/2,
  ]
  | *(
    f: 0|12 [2, 1],
    g: 16|16 [-1],
    l: 16|16 [-1],
    p: 5|5 [-1],
  ) Take 16
  | Seq [AsIs, Reverse | Lm 1/2]
  | Repeat 3
}`
}/>
