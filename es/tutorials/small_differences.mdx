Exploramos sonidos separados por grandes distancias como octavas. Ahora, jugaremos con sonidos que están muy juntos. A continuación, escuchará dos ondas sinusoidales separadas por 1 Hz.
Un hercio se define como un ciclo por segundo. Escuchará que los dos tonos crean un sonido oscilante con un ciclo de 1 segundo.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1, p: 0 }\n
main = {
  Overlay [
    Fa 1,
    Fa 0,
  ]
   -- Si descomentas la línea de abajo, puedes
   -- escuchar el sonido de latido se mantiene constante
   -- incluso cuando las frecuencias cambian.
  -- | Seq [Fm 1/2, Fm 1, Fm 2]
}`
}/>


Podemos hacer ritmos geniales construyendo armonías que aprovechen este sonido de pulsación. Escuche el siguiente intervalo, la proporción de 3/2 o una quinta perfecta, que también está imbuida del ritmo de 3 contra 2 gracias al uso de `Fa`.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1/2, p: 0 }\n
main = {
  Overlay [
    Fm 3/2, 
    Fm 3/2 | Fa 3,
    Fa 2,
    Fa 0,
  ]
}`
}/>

Suceden cosas interesantes cuando separas estas pequeñas diferencias en el espacio panoramizándolas. Recomiendo escuchar esto en auriculares o altavoces estéreo.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1/3, p: 0 }\n
main = {
  Overlay [
    Fm 9/4 | Fa 3 | Pa 1,
    Fm 9/4 | Pa 1,
    Fm 3/2 | Pa -1, 
    Fm 3/2 | Fa 2 | Pa -1,
    Fa 1 | Pa -1,
    Fa 0 | Pa 1,
    Fm 1/2 | Fa 1,
    Fm 1/2,
  ]
}`
}/>

## Un Acorde Genial
