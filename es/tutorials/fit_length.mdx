Una vez que hayas comenzado a escribir canciones con "Overlay", probablemente notarás que alinear los extremos de las frases puede ser tedioso.
Afortunadamente, las operaciones "FitLength" hacen que sea muy fácil de alinear
los fines de las operaciones.

## `Operación: FitLength`
Aquí, la operación2 es algo nuevo que has escrito.
operation1 es algo que escribiste anteriormente en la pieza.
`FitLength` va a estirar o aplastar la operación2
para que tenga la misma duración que la operación1.

<WereSoCool
  readOnly
  hideGutter
  language={`operation2 | FitLength operation1`}
/>

<WereSoCool language={
`{ f: 261.6, l: 1/7, g: 1, p: 0 }\n
-- Si escribimos esta melodía
melody = {
  Seq [
    Fm 2, Fm 15/8, Fm 3/2, Fm 5/3, Fm 3/2, Fm 5/4,
    Fm 9/8, Fm 1, Fm 5/6, Fm 1,
    Fm 3/4 | Lm 3
  ]
  | Seq [AsIs, Reverse]
}\n
main = {
  Overlay [
    -- Podríamos agregar una nota baja que estiraría
    -- a la duración de la melodía
    melody,
    Fm 1/2 
    -- Comente la operación FitLength para ver qué
    -- sucedería sin él.
    | FitLength melody
  ]
}`
}/>

Puedes crear relaciones rítmicas muy complicadas usando `Fitlength`.

<WereSoCool language={
`{ f: 440, l: 1/3, g: 1, p: 0 }\n
melody1 = {
  Overlay [
    {1/1, 3, 1/2, 1/2},
    {1/1, 0, 1/2, -1/2},
    {3/4, 0, 1/2, 0},
  ]
  | Seq [
    Fm 1, Fm 9/8, Fm 5/4, Fm 3/2, Fm 5/3 | Lm 3/2, 
    Seq [Fm 2, Fm 5/3] | Lm 1/2,
    Fm 3/2, Fm 1
  ]
}\n
melody2 = {
  Overlay [
    {1/2, 1, 1/2, -1},
    {1/2, 0, 1/2, 1}
  ]
  | Seq [
    Fm 3/4, Fm 5/6, Fm 5/4, Fm 2/3, Fm 5/4, Fm 9/4 
  ]
}\n
main = {
  Overlay [
    melody1,
    melody2 
    | Seq [Fm 1, Fm 7/8]
    | FitLength melody1
  ]
  | Repeat 3
}`
}/>
