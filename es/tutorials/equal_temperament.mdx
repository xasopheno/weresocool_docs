## `ET` -  Temperamento Igual

La operación ET crea una escala temperada igual dividida y por tiempos.
Esto da como resultado una lista en la que se puede indexar.

### ET(12)

<WereSoCool language={
`{ f: 261.6, l: 1/5, g: 1, p: 0 }\n
main = {
  -- Estas son las 12 notas que están disponibles en
  -- un instrumento musical occidental como un piano o
  -- una guitarra.
  ET(12) 
}`
}/>

### Temperamento Igual vs Armonía Basada en Proporciones

Un temperamento igual es un temperamento musical o sistema de afinación,
que aproxima solo intervalos dividiendo una octava
(u otro intervalo) en pasos iguales.
Esto significa la relación de las frecuencias de cualquier
par adyacente de notas es el mismo, lo que da un
el mismo tamaño de paso percibido que el tono se percibe aproximadamente
como el logaritmo de la frecuencia. - Wikipedia

La música occidental se escribe principalmente en temperamento igual de 12 tonos.
Dividimos y octavamos en 12 partes. El resultado es
una escala muy útil que aproxima - ciertamente no todo - pero
bastantes de los maravillosos intervalos que se pueden descubrir
en la serie armónica. Una pregunta importante es:
¿Qué tan buena es una aproximación? En algunos casos, ¡bastante bien!
En otros, puede ser útil, pero no es particularmente preciso.
Aquí hay una "escala cromática": todas las notas en temperamento igual de 12 tonos. Lo escucharás en entonación justa
(armonía basada en proporción) y temperamento igual al mismo tiempo.
Algunos de los intervalos son esencialmente los mismos
mientras que otros menos precisos.


<WereSoCool language={
`{ f: 440, l: 1, g: 1, p: 0 }\n
just = {
  Pa 1 |
  Seq [
    Fm 1, Fm 25/24, Fm 9/8, Fm 6/5, Fm 5/4, 
    Fm 4/3, Fm 7/5, Fm 3/2, Fm 8/5,
    Fm 5/3, Fm 7/4, Fm 15/8, Fm 2
  ]
}\n
equal = {
  Pa -1 | 
  Seq [
    ET(12) 
    @ [:, 0 | Fm 2]
  ]
}\n
main = {
  Overlay [
    just,
    equal
  ]
}`
}/>

### Temperamento extraño

Aunque de vez en cuando me quejo de la tiranía de los doce tonos
temperamento igual, temperamento igual es una herramienta muy útil.
WereSoCool permite la composición en cualquier temperamento. a menudo me gusta
componer en ET(19), ET(31), ET(72) y ET(120).

ET(120) es interesante porque divide cada semitono en
Temperamento igual de 12 tonos en 10 partes... ¡vaya!
Eso facilita que los músicos entrenados piensen en dodecafonía.
temperamento igual para explorar algunas melodías bastante expresivas,
sin dejar de pensar de manera tradicional. Independientemente de sus antecedentes,
Recomiendo experimentar con componer melodías en todo tipo de temperamentos iguales.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
overtones = {
  O[
    (1/1, 0, 1, 0),
    (1/2, 3, 1, -1/2),
    (1/2, 0, 1, 1/2),
    (1/4, 2, 1/2, 1/2),
  ]
}\n
-- ET puede considerarse simplemente como un caso especial de la lista.
-- Puede crear listas en el nivel superior y reutilizarlas.
three_tone_et = ET(3)
seven_tone_et = ET(7)
twelve_tone_et = ET(12)
seventeen_tone_et = ET(17)\n
all_scales = {
  overtones |
  Seq [
    -- Cuando quiera usar una lista más adelante, antepóngala con
    -- un símbolo &.
    &three_tone_et,
    &seven_tone_et | Reverse,
    &twelve_tone_et | Reverse,
    &seventeen_tone_et
  ]
}\n
weird_microtonal_melody = {
   O[
    (2, 3, 1, -1),
    (2, 0, 1, 1),
    (3/2, 3, 1, -1),
    (3/2, 0, 1, 1),
    (1/1, 0, 1, 0),
    (1/2, 0, 1, 0),
  ]
  | ET(120) 
    @ [
      47, 80, 55, 50, 20, 30, 55, 61, 
      12, 62, 77, 64, 10, 44, 32, 
      31, 22, 11, 25, 77, 90, 20, 70, 
      70, 74, 70, 73, 0, 30, 70, 65, 73, 74,
      70, 74, 90, 100, 110, 74, 70, 0, 20, 40
    ]
    | Fm 2 
    | Gm 1/2
    | FitLength all_scales
}\n
main = {
  Overlay [
    all_scales,
    Seq [
      weird_microtonal_melody
    ]
  ]
  | Seq [AsIs, Reverse]
  | Lm 1/8
  | Gm 1/3
}`
}/>
