Antes de continuar, podemos agregar un par de operaciones más a nuestro repertorio.

## `Reverse`
Reverse toma una entrada y la reproduce hacia atrás.

<WereSoCool language={
`{ f: 261.6, l: 1/6, g: 1, p: 0 }\n
escala_mayor = {
   Seq [
     Fm 1, Fm 9/8, Fm 5/4, Fm 4/3,
     Fm 3/2, Fm 5/3, Fm 15/8, Fm 2
   ]
}\n
main = {
   Seq [
     escala_mayor,
     escala_mayor | Reverse
   ]
}`
}/>

## `Repeat <número>`
Repeat es genial si quieres que algo vuelva a suceder.

<WereSoCool language={
`{ f: 261.6, l: 1/4, g: 1, p: 0 }\n
melodia = {
  Seq [Fm 5/4, Fm 9/8, Fm 5/4, Fm 1]
}\n
main = {
  Seq [
    melodia | Repeat 3
  ]
}`
}/>

# Seq | Seq 

Si ha estado siguiendo, es posible que se pregunte,
¿Qué sucede cuando haces algo como esto?:

`Seq [op1, op2, ...] | Seq [op3, op4, ...]`

<WereSoCool language={
`{ f: 220, l: 1, g: 1, p: 0 }\n
-- Necesitamos un sonido genial para explorar esta nueva idea.
-- Esta es una escala_pentatónica. Es un subconjunto genial de escala mayor.
escala_pentatonica = {
   Seq [
       Fm 1, Fm 9/8, Fm 5/4,
       Fm 3/2, Fm 5/3
   ]
   -- 8 veces más rápido... ¡wiiiiii!
   | Lm 1/8
}\n
main = {
   -- Ahora conectaremos la escala pentatónica a la secuencia [Fm 1, Fm 9/8, Fm 5/4].
   -- El resultado será la escala pentatónica subiendo en tono entero
   escala_pentatonica
   | Seq [
       Fm 1, Fm 9/8, Fm 5/4,
       -- ===== descomente la siguiente línea con cmd + / (ctl + / en linux/windows)======
       -- Fm 9/8 | Reverse
       -- El reverso es genial. Toma lo que sea que le pongas
       -- y lo invierte. Simple como eso.
   ]
   -- ===== y descomente la siguiente línea con cmd + / (ctl + / en linux/windows)
   -- | Repeat 4
}`
}/>
