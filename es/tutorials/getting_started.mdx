**<CoolText>WereSoCool</CoolText>** es un lenguaje de programación para componer música microtonal. Puede encontrar este idioma útil si desea hacer sonidos geniales e impresionar a sus amigos/mascotas/plantas. Este lenguaje no requiere familiaridad ni con la música microtonal ni con la programación informática, pero la experiencia con cualquiera de los dos sin duda ayudará.

Mi enfoque recomendado para aprender el idioma es hacer los tutoriales en orden y escribir su propia composición después de completar cada uno.

**WereSoCool CLI** está disponible en **macOS**, **Linux** y **Windows**. También puede explorar el idioma en los navegadores Firefox y Chrome en **<CoolLink href={"/playground"}>WereSoCool Playground</CoolLink>**.

<Break/>

## WereSoCool Playground:

Puedes hacer sonidos geniales en los navegadores Firefox y Chrome en el **<CoolLink href={"/playground"}>WereSoCool
Playground</CoolLink>**. WereSoCool tiene un solo subproceso en el navegador, por lo que es probable que las composiciones más complicadas no se rendericen sin hacer clic. El WereSoCool Playground aún no se ejecuta en dispositivos móviles.

## Instalación de la CLI de WereSoCool

### Cargo:

WereSoCool se puede instalar en macOS, Linux y Windows a través de Cargo. También deberá instalar las dependencias opcionales para usar algunas funciones.

Cargo es el "package manager" para Rust. Instale Cargo instalando <CoolLink href={"https://www.rust-lang.org/en-US/install.html"}>Rustup</CoolLink>. Después de instalar Cargo, ejecute:

`cargo install weresocool`

#### Dependencias opcionales
Si desea renderizar a mp3 y oggvorbis, necesitará las siguientes dependencias

```
Macos:
brew install lame libvorbis

Arch Linux:
sudo pacman -S lame vorbis-tools

Ubuntu:
sudo apt-get lame vorbis-tools
```

### macOS:
La CLI de WereSoCool está disponible en macOS a través de Homebrew. Si no tiene Homebrew instalado, siga las instrucciones de instalación <CoolLink href={"https://brew.sh/"}>aquí</CoolLink>.

Instale WereSoCool desde Homebrew ejecutando los siguientes comandos:

`brew tap xasopheno/weresocool && brew install weresocool`

Este software se ejecuta en Windows con una capacidad limitada. En Windows, este software está compilado sin compatibilidad con mp3 y oggvorbis (solo puede imprimir en wav, csv y json). ## Editor Support
Algunos editores comunes tienen disponible un resaltado de sintaxis simple. Si desea crear un complemento para su editor favorito, contáctame.

<Break/>

### Vim
Mi editor de elección actualmente es NeoVim, por lo que el soporte en vim está por delante de otros editores. Disponible en <CoolLink href={"https://github.com/xasopheno/WereSoCool_vim"}>Github</CoolLink>.

Si usa vim-plug, agregue la siguiente línea `init.vim` y ejecute `:PlugInstall`

`Plug 'xasopheno/weresocool_vim'`

<Break/>

### VisualStudio Code
El resaltado de sintaxis mínima está disponible a través de la extensión weresocool. Busque 'weresocool' en las extensiones o <CoolLink href={"https://marketplace.visualstudio.com/items?itemName=weresocool.socool"}> instálelo desde el mercado.</CoolLink>
<Break/>

### SublimeText
El resaltado de sintaxis mínima está disponible a través del paquete weresocool. Puede encontrarlo en <CoolLink href={"https://github.com/xasopheno/weresocool_sublimetext"}>Github</CoolLink>.
