Si te divertiste tanto haciendo acordes geniales como yo, te alegrará saber que hay una abreviatura para hacer precisamente eso.

## `Operación: Overtone` 
<WereSoCool
  readOnly
  hideGutter
  language={
`-- Cada conjunto de cuatro valores describe
-- la frecuencia se multiplica, la frecuencia se suma,
-- ganar multiplicar y pan agregar de una nota.
Overlay[(Fm, Fa, Gm, Pa), ...]`}
/>
<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
-- Por ejemplo: Esta Overlay[] describe dos notas.
-- El fondo es lo fundamental - 1/1
-- en ganancia 1 panoramizó 1/2 de la derecha y
-- la parte superior es una nota una octava por encima de la fundamental
-- más 2hz en ganancia 1 panoramizado 1/2 a la izquierda.
-- Generalmente organizo las notas con la nota más baja en la parte inferior.
  Overlay [
    {2/1, 2, 1, 1/2},
    {1/1, 0, 1, -1/2},
  ]
}`
}/>


Dentro de la Overlay[], puedes poner tantos (Fm, Fa, Gm, Pa) como quieras.
esta puede ser una excelente manera de componer acordes de varias voces en una serie
o, como se usa en la siguiente canción, para agregar sobretonos a otras operaciones.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
overtones = {
-- Overlay[(Fm, Fa, Gm, Pa), ...]
  Overlay [
    {3/1, 1, 1/4, -1/4},
    {3/1, 0, 1/4, 1/4},
    {1/1, 2, 1, 1/2},
    {1/1, 0, 1, -1/2},
  ]
}\n
-- Esta es una pequeña canción de dos partes:
melody1 = {
  Seq [
    Fm 1, Fm 9/8, Fm 5/4 | Lm 2, Fm 9/8, Fm 1,
    Fm 3/2 | Lm 4, Fm 5/3, Fm 9/8
  ]
}\n
melody2 = {
  Seq [
    Fm 5/4, Fm 3/2, Fm 5/3 | Lm 2, Fm 3/2, Fm 4/3,
    Fm 9/8 | Lm 3/2, Fm 5/4 | Lm 3/2, Fm 4/3 | Lm 3 
  ]
  | Fm 1/2
}\n
main = {
-- "overtones" contiene 4 voces
-- definido arriba con la sintaxis Overlay[].
-- conectándolo a la superposición
-- da como resultado el mismo sonido que
-- si escribimos lo siguiente:
-- Overlay [
--   overtones | melody1,
--   overtones | melody2
--]
  overtones
  | Overlay [
    melody1,
    melody2,
  ]
-- Luego tocamos las melodías de arriba primero en Fm 1
-- (donde normalmente jugarían),
-- luego reprodúzcalo hacia atrás usando Reverse.
  | Seq [
    Fm 1, 
    Reverse
  ]
-- Más rápida
  | Lm 2/5
-- 2x
  | Repeat 2 
-- Mas tranquila
  | Gm 1/3
}
`
}/>
