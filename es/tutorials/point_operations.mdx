Tomemos un momento para ver todas las `Operaciones de puntos`. Estas operaciones son la base de todas las demás operaciones.

## `Fm <número> - Multiplicación de frecuencia`

- Multiplica la frecuencia de entrada por el número proporcionado.
- El valor puede ser cualquier Número >= 0.
- Valores muy altos o muy bajos pueden no ser audibles.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Fm 5/4
}`
}/>

## `Fa <número> - Adición de frecuencia`

- Suma el número proporcionado a la frecuencia de entrada.
- El valor puede ser cualquier número.
- Si la suma de la frecuencia de entrada y el número proporcionado es inferior a 0, el sonido se silenciará.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Fa -7
}`
}/>

## `Lm <número> - Multiplicación de longitud`

- Multiplica la longitud de entrada por el número proporcionado.
- El valor puede ser cualquier Número >= 0.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Lm 3
}`
}/>

## `Gm <número> - Multiplicación de gain`

- Multiplica la ganancia de entrada por el número proporcionado.
- El valor puede ser cualquier Número >= 0.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Gm 1/2
}`
}/>

## `Pm <número> - Multiplicación de pan`

- Multiplica el pan de entrada por el número proporcionado.
- El valor puede ser cualquier número.
- Se recomienda mantener los valores de pan entre -1 y 1
- Los números positivos se desplazan hacia la izquierda y los negativos hacia la derecha.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: -1 }\n
main = {
  Pm 7/8
}`
}/>

## `Pa <number> - Adición de pan`

- Agrega el número proporcionado a la bandeja de entrada.
- El valor puede ser cualquier número.
- Se recomienda mantener los valores de pan entre -1 y 1
- Los números positivos se desplazan hacia la izquierda y los negativos hacia la derecha.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Pa 9/10 
}`
}/>

## Operaciones de identidad
- Algunas operaciones son "no-ops" que actúan como una identidad.
  - Fm 1, Gm 1, Lm 1, Pm 1, Fa 0 y Pa 0 son operaciones de identidad.
