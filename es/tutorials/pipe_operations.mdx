## `Operación: operación | operación - Pipe`
Una de las operaciones más importantes en este lenguaje es "|" (Pipe),
porque te permite encadenar operaciones.


<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
   Seq [
     -- Las operaciones son acumulativas, entonces:
     -- Fm 3/2 | Fm 5/4 == Fm 15/8
     Fm 1,
     Fm 3/2 | Fm 5/4,
     Fm 1,
     Fm 15/8
   ]
}`
}/>

<WereSoCool language={
`{ f: 261.6, l: 1/3, g: 1, p: 0 }\n
-- Podemos factorizar nuestra escala mayor en su propio bloque.
escala_mayor = {
  Seq [
         Fm 1, Fm 9/8, Fm 5/4, Fm 4/3,
         Fm 3/2, Fm 5/3, Fm 15/8, Fm 2
   ]
}\n
main = {
   Seq [
     -- Podemos usar el bloque de escala mayor por nombre aquí.
     escala_mayor,
     -- Ahora podemos usar la operación de tubería para reproducir una escala mayor un paso y el doble de rápido
     escala_mayor
     -- multiplique todas las frecuencias en nuestra escala mayor por 9/8 para subir un paso.
     | Fm 9/8
     -- multiplicar toda la longitud de todas las notas en nuestro
     -- escala_mayor por 1/2
     | Lm 1/2
   ]
}`
}/>
