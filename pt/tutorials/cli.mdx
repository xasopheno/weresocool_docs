Depois de instalar a WereSoCool CLI, você pode fazer muitas coisas legais:

## Demo

Todo software estranho deve ter uma boa demonstração. Se você quiser fazer um som legal e impressionar seus amigos/bichos/plantas sem fazer nenhum trabalho duro, você pode simplesmente executar:

`weresocool demo` 

Esta é uma boa maneira de verificar se WereSoCool está funcionando corretamente em seu sistema. Se você não gostar do som que ele faz, digite `ctrl-c` para parar o aplicativo. Você sempre pode usar `ctrl-c` para parar o aplicativo.

<Break/>

## New

Agora que você ouviu um som legal, você pode querer fazer um som legal de sua preferência. A maneira mais fácil de fazer isso é executar:

`weresocool new [o_nome_legal_da_sua_composição_legal]` 

Isso criará um novo arquivo `.socool` usando o modelo WereSoCool.. 

<Break/>

## Play

Se você quiser ouvir uma composição escrita em WereSoCool, você pode executar:

`weresocool play [filepath]`

Isso tocará a música uma vez.

<Break/>

Quando você está desenvolvendo uma música, você vai querer usar o flag `--watch`:

`weresocool play [filepath] --watch`

Isso repetirá a composição toda vez que o arquivo for salvo.

Quando estou compondo, o que normalmente faço é `weresocool play [composition] --watch` o arquivo que estou trabalhando em um terminal lado a lado com a composição em meu editor.

<Break/>

## Print

Você pode renderizar um arquivo `.socool` para vários alvos diferentes. O padrão é um arquivo `.wav`.

`weresocool print [filename]` renderizará essa composição em um arquivo wav.

Você pode, opcionalmente, passar vários outros flags.
  Usage: weresocool print [OPTIONS] [filename]
  Options:
      --output_dir               [output_dir]
      --wav                      imprimir um arquivo wav (padrão)
      --mp3                      imprimir um mp3
      --oggvorbis                imprimir um oggvorbis arquivo
      --csv                      imprimir um csv arquivo
      --json                     imprimir um json arquivo
      --stems                    imprimir stems como um arquivo zip
      --sound                    imprimir todos os tipos de arquivo de som
      --all                      imprimir todos os tipos de arquivo

<Break/>

## Help

Você sempre pode usar o sinalizador de ajuda quando ficar preso. Infelizmente, a ajuda será impressa em inglês.

`weresocool --help` imprimirá o seguinte:

```
Usage: weresocool [COMMAND]

Commands:
  new    Create a new .socool file from the template
  play   Render a .socool file. Run with --watch to rerend on file save
  demo   Hear a cool sound
  print  Print a .socool composition to a file
  help   Print this message or the help of the given subcommand(s)
```
