Até agora, exploramos apenas os intervalos melodicamente. Agora, adicionaremos uma nova Operação - Overlay - que nos permitirá explorar intervalos harmonicamente.


## `Operação: Overlay` 
<WereSoCool
  readOnly
  hideGutter
  language={`Overlay [operações]`}
/>

O Overlay pega uma série de operações e as executa ao mesmo tempo. 

<WereSoCool language={
`{ f: 261.6, l: 3, g: 1, p: 0 }\n
main = {
  -- Tríade Dó Maior. Uau, Aaaah.
  Overlay [
    Fm 3/2,
    Fm 5/4,
    Fm 1, 
  ]
}`
}/>

Que tal uma sequência de sobreposições?

## Exploração Mais Harmônica

Você verá um BeloGráfico™ à direita do seguinte o WereSoCool Editor (abaixo no celular). Você pode usar este gráfico para se orientar na rica paisagem da harmonia baseada em proporção. Proporções cercadas por Caixas Douradas são as proporções que descrevem a escala Maior. Ratios of similar dissonace are the same color. Proporções de dissonâncias semelhantes são da mesma cor. Há muitos novos intervalos para explorar dos quais você talvez nunca tenha ouvido falar. Terceiro submenor? Sim por favor. 11/8 Trítono? muito legal.
