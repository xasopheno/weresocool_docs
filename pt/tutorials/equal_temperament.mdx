## `Operação: ET - Temperamentos Iguais` 
A operação ET cria uma escala temperada igual dividida `n` vezes.
Isso resulta em uma lista que pode ser indexada.

### ET(12)

<WereSoCool language={
`{ f: 261.6, l: 1/5, g: 1, p: 0 }\n
main = {
   -- Estas são as 12 notas que estão disponíveis no
   -- um instrumento musical ocidental, como um piano ou
   -- um violão.
  ET(12) 
}`
}/>

### Temperamento Igual vs Harmonia Baseada em Razão

Um temperamento igual é um temperamento musical ou sistema de afinação, que aproxima intervalos justos dividindo uma oitava
(ou outro intervalo) em partes iguais.
Isso significa que a razão das frequências de qualquer
par adjacente de notas é o mesmo, o que dá uma
tamanho do passo percebido igual à medida que o tom é percebido aproximadamente como o logaritmo da frequência. - Wikipédia

A música ocidental é principalmente escrita em temperamento igual de 12 tons. Nós dividimos e oitava em 12 partes. O resultado é
uma escala muito útil que aproxima - certamente não todos - mas
alguns dos maravilhosos intervalos detectáveis
na série harmônica. Uma pergunta importante é:
Quão boa é uma aproximação? Em alguns casos, muito bom!
Em outros, pode ser útil, mas não é particularmente preciso.
Aqui está uma 'escala cromática' - todas as notas em temperamento igual de 12 tons. Você vai ouvir em entonação justa
(harmonia baseada na razão) e temperamento igual ao mesmo tempo.
Alguns dos intervalos são essencialmente os mesmos
enquanto outros menos precisos.


<WereSoCool language={
`{ f: 440, l: 1, g: 1, p: 0 }\n
just = {
  Pa 1 |
  Seq [
    Fm 1, Fm 25/24, Fm 9/8, Fm 6/5, Fm 5/4, 
    Fm 4/3, Fm 7/5, Fm 3/2, Fm 8/5,
    Fm 5/3, Fm 7/4, Fm 15/8, Fm 2
  ]
}\n
equal = {
  Pa -1 | 
  Seq [
    ET(12) 
    @ [:, 0 | Fm 2]
  ]
}\n
main = {
  Overlay [
     justo,
     igual
  ]
}`
}/>

### Temperamento Estranho

Embora eu possa reclamar de vez em quando sobre a tirania de 12 tons temperamento igual, temperamento igual é uma ferramenta muito útil.
WereSoCool permite composição em qualquer temperamento. Muitas vezes eu gosto de
compor em ET(19), ET(31), ET(72) e ET(120).

ET(120) é interessante porque divide cada meio passo em
Temperamento igual de 12 tons em 10 partes... uau!
Isso torna mais fácil para músicos treinados para pensar em 12 tons
temperamento igual para explorar algumas melodias bastante expressivas,
enquanto ainda pensa de maneira tradicional. Independentemente do seu passado,
Eu recomendo experimentar compor melodias em todos os tipos de temperamentos iguais.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n

  Overlay[
    {1/1, 0, 1, 0},

  ]
}\n
-- ET pode ser pensado simplesmente como um caso especial da lista.
-- Você pode criar listas no nível superior e reutilizá-las.
three_tone_et = ET(3)
seven_tone_et = ET(7)
twelve_tone_et = ET(12)
seventeen_tone_et = ET(17)\n
all_scales = {
  overtones |
  Seq [
    -- Quando você quiser usar uma lista mais tarde, você a precede com
     -- um símbolo &.
    &three_tone_et,
    &seven_tone_et | Reverse,
    &twelve_tone_et | Reverse,
    &seventeen_tone_et
  ]
}\n
estranha_melodia_microtonal = {
   Overlay[
    {2, 3, 1, -1},
    {2, 0, 1, 1},
    {3/2, 3, 1, -1},
    {3/2, 0, 1, 1},
    {1/1, 0, 1, 0},
    {1/2, 0, 1, 0},
  ]
  | ET(120) 
    @ [
      47, 80, 55, 50, 20, 30, 55, 61, 
      12, 62, 77, 64, 10, 44, 32, 
      31, 22, 11, 25, 77, 90, 20, 70, 
      70, 74, 70, 73, 0, 30, 70, 65, 73, 74,
      70, 74, 90, 100, 110, 74, 70, 0, 20, 40
    ]
    | Fm 2 
    | Gm 1/2
    | FitLength all_scales
}\n
main = {
  Overlay [
    all_scales,
    Seq [
      estranha_melodia_microtonal 
    ]
  ]
  | Seq [AsIs, Reverse]
  | Lm 1/8
  | Gm 1/3
}`
}/>
