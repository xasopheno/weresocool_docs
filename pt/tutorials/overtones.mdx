## `Operação: Seq`
A próxima operação mais útil é a operação Sequence ou Seq.

Podemos usar `Seq` para criar uma série de sons. Estas são as primeiras 12 notas da série de harmônicos começando em 261,6 hz - um "C" no piano.

## Descobrindo Intervalos

Há intervalos que aparecem na música em todo o mundo. De onde eles vêm? Eles vêm de uma estrutura na natureza que é chamada de série harmônica na qual cada frequência é um múltiplo inteiro de uma fundamental. Vamos explorar a série harmônica e encontrar alguns sons antigos e novos.

### Complexidade do intervalo

Os intervalos são nomeados em relação ao fundamental. Neste caso, a fundamental pode ser descrita pela operação `Fm 1`. O intervalo simples é uníssono. Fm 1 soando ao mesmo tempo que Fm 1. Seguindo o uníssono é a duplicação (Fm 2) e metade das frequências (Fm 1/2) que vimos no último tutorial. Uma frequência dobrada é conhecida como uma oitava na teoria musical ocidental (TMO). Outra maneira de dizer que a distância entre o 1º e o 2º harmônico é uma oitava.

E a distância entre o 2º e o 3º sobretons? O nome em TMO é o `Perfect 5th`. Ouça abaixo o intervalo. Primeiro na posição natural na série harmônica e depois a razão transposta uma oitava abaixo para que possamos ouvi-la no contexto da fundamental, `Fm 1`.

<WereSoCool language={
`{ f: 261.6, l: 1, g: 1, p: 0 }\n
main = {
  Seq [
  -- Primeiro em sua posição natural na série harmônica.
    Fm 2, 
    Fm 3, 
    Fm 2, 
  -- E agora usaremos o intervalo diretamente -- 3/2 -- que soará em
  -- a mesma oitava que a fundamental.    
    Fm 1, 
    Fm 3/2, 
    Fm 1, 
  ]
}`
}/>

### Escala Maior

Podemos descobrir todos os tipos de intervalos bonitos e úteis explorando as distâncias entre as séries harmônicas naturais.


Aqui está uma escala de dó maior (261,6 hz é um lá) construída usando proporções da série de harmônicos. A primeira nota é a fundamental. A segunda é a distância entre os tons 9 e 8. O terceiro intervalo é a distância entre o 5º e o 4º harmônico...etc...

<WereSoCool language={
`{ f: 261.6, l: 1/3, g: 1, p: 0 }\n
main = {
  Seq [
    Fm 1, Fm 9/8, Fm 5/4, Fm 4/3, 
    Fm 3/2, Fm 5/3, Fm 15/8, Fm 2
  ]
}`
}/>

### Faça Uma Melodia

Agora podemos criar todos os tipos de melodias maravilhosas e descobrir novos intervalos legais. Reserve um momento para escrever sua própria melodia.

`{ f: 261.6, l: 1/7, g: 1, p: 0 }\n
main = {
  Seq [
    Fm 1, Fm 5/4, Fm 3/2, Fm 5/3, 
    Fm 7/4, Fm 2, Fm 3/2, Fm 9/8, 
    
    Fm 1, Fm 25/24, Fm 9/8, Fm 6/5, 
    Fm 5/4, Fm 3/2, Fm 15/8, Fm 3/2,
    
    Fm 2, Fm 9/8, Fm 4/3, Fm 5/3, 
    Fm 3/2, Fm 4/3, Fm 5/4, Fm 9/8, 
    
    Fm 1, Fm 5/4, Fm 3/2, Fm 2, Fm 3/2, Fm 5/4, Fm 1/2
  ]
}`
}/>
