Timbre operations change the waveform generated by an oscillator. Be careful when moving from one timbre to and other in the same voice, it won't always work well. It's best to either create a new voice or add a space when you want to change oscillator type.

## `Operação: Sine` 

A operação `Sine` aplica a seguinte fórmula para produzir a forma de onda. `Sine 1` é o som padrão no WereSoCool.

`sin(fase^potência) / potência`


<WereSoCool
  readOnly
  hideGutter
  language={`
  -potência precisa ser um valor maior que 0.\n
  Sine <potência>`}
/>


<WereSoCool language={
`{ f: 161.6, l: 2, g: 1/2, p: 0 }\n
main = {
  Seq [Fm 1, Fm 0 | Lm 1/4]
  | Seq [
    Sine 1, 
    Sine 1.5, 
    Sine 2.0, 
    Sine 2.5, 
    Sine 3, 
    Sine 3.5,
    Sine 0.5,
    Sine 1, 
  ]
}`
}/>

## `Operação: Tri` 

A operação `Tri` aplica a seguinte fórmula a cada amostra.

`(abs(sin(fase^potência)) * 2 - 1) / potência` 

<WereSoCool
  readOnly
  hideGutter
  language={`--potência precisa ser um valor maior que 0.\n
  Tri <power>`}
/>


<WereSoCool language={
`{ f: 161.6, l: 2, g: 1/2, p: 0 }\n
main = {
  Seq [Fm 1, Fm 0 | Lm 1/4]
  | Seq [
    Tri 1, 
    Tri 1.5, 
    Tri 2.0, 
    Tri 2.5, 
    Tri 3, 
    Tri 3.5,
    Tri 0.5,
    Tri 1, 
  ]
}`
}/>

## `Operação: Square` 

O `Square` pode ser pensado como uma operação de 'pulse width' de pulso. O valor em qualquer amostra é calculado com o
Fórmula:

s = sin(fase);
de a > pulse width { -1. } else { 1. };

O pulse_width padrão é 0,5

<WereSoCool
  readOnly
  hideGutter
  language={`--O pulse_width precisa ser um valor > 0 e < 1\n
  Square <pulse_width>`}
/>


<WereSoCool language={
`{ f: 161.6, l: 2, g: 1/2, p: 0 }\n
main = {
  Seq [Fm 1, Fm 0 | Lm 1/4]
  | Seq [
    Square 0.1, 
    Square 0.5, 
    Square 0.75, 
  ]
}`
}/>
