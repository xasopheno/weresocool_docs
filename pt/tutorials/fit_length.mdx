Depois de começar a escrever músicas com sobreposições, você provavelmente notará
que alinhar as extremidades das frases pode ser tedioso.
Felizmente, as operações "FitLength" facilitam o alinhamento
o fim das operações.

## `Operação: FitLength`
Aqui, a operação2 é algo novo que você escreveu.
operação1 é algo que você escreveu anteriormente na composição.
FitLength vai esticar ou esmagar a operação2
para torná-lo a mesma duração da operação1.

<WereSoCool
  readOnly
  hideGutter
  language={`operação2 | FitLength operação1`}
/>

<WereSoCool language={
`{ f: 261.6, l: 1/7, g: 1, p: 0 }\n
-- Se escrevêssemos esta melodia
melody = {
  Seq [
    Fm 2, Fm 15/8, Fm 3/2, Fm 5/3, Fm 3/2, Fm 5/4,
    Fm 9/8, Fm 1, Fm 5/6, Fm 1,
    Fm 3/4 | Lm 3
  ]
  | Seq [AsIs, Reverse]
}\n
main = {
  Overlay [
    -- Poderíamos adicionar uma nota baixa que esticaria
    -- para a duração da melodia
    melody,
    Fm 1/2 
    -- Comente a operação FitLength para ver o que
    -- aconteceria sem ele.
    | FitLength melody
  ]
}`
}/>

Você pode criar relações rítmicas muito complicadas usando Fitlength.

<WereSoCool language={
`{ f: 440, l: 1/3, g: 1, p: 0 }\n
melody1 = {
  Overlay [
    {1/1, 3, 1/2, 1/2},
    {1/1, 0, 1/2, -1/2},
    {3/4, 0, 1/2, 0},
  ]
  | Seq [
    Fm 1, Fm 9/8, Fm 5/4, Fm 3/2, Fm 5/3 | Lm 3/2, 
    Seq [Fm 2, Fm 5/3] | Lm 1/2,
    Fm 3/2, Fm 1
  ]
}\n
melody2 = {
  Overlay [
    {1/2, 1, 1/2, -1},
    {1/2, 0, 1/2, 1}
  ]
  | Seq [
    Fm 3/4, Fm 5/6, Fm 5/4, Fm 2/3, Fm 5/4, Fm 9/4 
  ]
}\n
main = {
  Overlay [
    melody1,
    melody2 
    | Seq [Fm 1, Fm 7/8]
    | FitLength melody1
  ]
  | Repeat 3
}`
}/>
