Exploramos sons separados por grandes distâncias como oitavas. Agora, vamos brincar com sons muito próximos. Abaixo, você ouvirá duas ondas senoidais separadas por 1 hz.
Um hertz é definido como um ciclo por segundo. Você ouvirá os dois tons criando um som de batida com um ciclo de 1 segundo.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1, p: 0 }\n
main = {
  Overlay [
    Fa 1,
    Fa 0,
  ]
   -- Se você descomentar a linha abaixo, você pode
   -- ouvir o som da batida permanece consistente
   -- mesmo quando as frequências mudam.
   -- | Seq [Fm 1/2, Fm 1, Fm 2]
}`
}/>


Podemos fazer ritmos legais construindo harmonias que tirem proveito desse som de batida. Ouça o seguinte intervalo - a proporção de 3/2 ou uma 5ª perfeita - que também está imbuída do ritmo de 3 contra 2.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1/2, p: 0 }\n
main = {
  Overlay [
    Fm 3/2, 
    Fm 3/2 | Fa 3,
    Fa 2,
    Fa 0,
  ]
}`
}/>

Coisas interessantes acontecem quando você separa essas pequenas diferenças no espaço fazendo uma panorâmica. Recomendo ouvir em fones de ouvido.

<WereSoCool language={
`{ f: 261.6, l: 5, g: 1/3, p: 0 }\n
main = {
  Overlay [
    Fm 9/4 | Fa 3 | Pa 1,
    Fm 9/4 | Pa 1,
    Fm 3/2 | Pa -1, 
    Fm 3/2 | Fa 2 | Pa -1,
    Fa 1 | Pa -1,
    Fa 0 | Pa 1,
    Fm 1/2 | Fa 1,
    Fm 1/2,
  ]
}`
}/>
