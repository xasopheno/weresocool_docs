## `Operação: import` 

Você pode importar composições de outros arquivos `.socool`.

O comportamento padrão é usar o nome do arquivo .socool como nome base para a importação, mas você pode usar a sintaxe `as` para renomear a importação.

Depois de importar o arquivo, você pode usar a notação de ponto para acessar a parte da composição importada que deseja usar.
